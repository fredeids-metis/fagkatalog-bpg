# Fagkatalog

Interaktiv fagkatalog for videregående skole. Viser alle programfag med beskrivelser, bilder og video.

## Bruk

Åpne `index.html` direkte i nettleser, eller se embed.html for Squarespace-innbygging.

## Datakilde

All fagdata hentes fra [school-data](https://github.com/fredeids-metis/school-data) API:

```
https://fredeids-metis.github.io/school-data/api/2025-01/curriculum/fag.json
```

## Filer

- `index.html` - Standalone side
- `embed.html` - Squarespace embed-kode
- `app.js` - Hovedlogikk
- `style.css` - Styling
